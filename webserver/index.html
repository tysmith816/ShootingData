Publicly available shooting victim data for 2024-04-03:

Data obtained from OpenDataPhilly's Shooting Victims API.

This page can present data from 2015-01-01 through 2024-12-31. See OpenDataPhilly for more information.



Information about the creation of this page:

Back end:

The back end of this page is a node express server that routes get commands from the front end.

The get request includes a requested date. A fetch command is executed to retrieve a JSON table

from OpenDataPhilly's public API (the URL includes a SQL command). The retrieved data is filtered

for the requested date and then returned to the front end in response to the request.


Front end:

The front end of this page is a node HTTP server that serves an index.html page formatted by a css

stylesheet. The HTML page has a date input field which triggers a request to the back end when the

selected date changes. The response from the back end is then filtered and formatted into a table.
